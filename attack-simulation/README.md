# Attack Simulation

Laboratórios para simular ataques comuns e depois analisá-los.

## Objetivo

Gerar atividades maliciosas controladas para estudar:

- detecção de ameaças
- análise de logs
- resposta a incidentes

## Tipos de simulação

- Port scan
- Tentativa de brute force
- Reconhecimento de rede

## Ferramentas usadas

- Nmap
- Linux logs
- análise de rede
