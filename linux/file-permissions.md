# Linux File Permissions

Permissões controlam quem pode ler, escrever ou executar arquivos no Linux.

## Tipos de permissão

r = read (ler)

w = write (escrever)

x = execute (executar)

## Exemplo

chmod 755 arquivo.sh

Significado:

Owner: read, write, execute  
Group: read, execute  
Others: read, execute
